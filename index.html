Setup Development Dependences:

For 64 bit Systems

. sudo apt-get install git-core gnupg flex bison gperf libsdl-dev libesd0-dev   
. sudo apt-get libwxgtk2.6-dev build-essential zip curl libncurses5-dev zlib1g-dev  
. suto apt-get sun-java5-jdk 

Create Repo Command and point path to run it:

. cd ~ 
. mkdir bin 
. sudo gedit .bashrc 

Insert the following at the bottom of the text file that opens.
export PATH=${PATH}:~/bin 
Save and Exit

. curl http://android.git.kernel.org/repo >~/bin/repo 
. chmod a+x   ~/bin/repo

Obtaining Source:

$ mkdir android
$ mkdir system
$ cd android/system
$ repo init -u https://github.com/MotoMania/MotoMania_Manifest -b master 
$ repo sync

Building the System:

. ./build/envsetup.sh  
. lunch
. make -j8 otapackage CND_RELEASE=true
